Then & Now Film

"Then & Now" is a video project we created in first year. This project is based on how communication has changed over time. Before we had to write letter to people and wait for days to get one response. Meanwhile now we can send a message and get a reply back within seconds using the technology now a days. This project was our first videography project where we got the chance to expand our knowledge on masked transitions, colour grading and story telling.
